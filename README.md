# Task 3: HR Employee Attrition Dashboard

##  Objective
Create an interactive HR Employee Attrition Dashboard using Microsoft Power BI to analyze employee turnover and identify the key factors contributing to attrition. The dashboard is designed to answer the business question, **"Why are employees leaving?"**, through interactive filters, KPI cards, and insightful visualizations.

##  Dataset
- HR Employee Attrition Dataset (SkillCraft Technology)

## 🛠 Tools Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)

##  Dashboard Features
- # KPI Cards
- Total Employees
- Attrition Count
- Attrition Rate
- # Interactive Slicers 
- Department
-  Age Group
-  Business Travel(region)
- Attrition by Department
- Attrition by Age Group
- Attrition by Job Role
- Attrition by Gender
- Key Influencers Visual
- Interactive Dashboard with Dynamic Filtering

## Key Insights
- Analyzed employee attrition across departments, age groups, and job roles.
- Identified departments and job roles with the highest attrition.
- Evaluated the impact of business travel and gender on employee turnover.
- Used the Key Influencers visual to determine the primary factors associated with employees leaving the organization.
- Built an interactive dashboard to support HR analytics and decision-making.

Successfully developed an interactive HR Employee Attrition Dashboard that provides meaningful insights into **why employees are leaving**.
